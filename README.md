# SerialMagnet
SerialBox gem for wrapping the content module of 29th street

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'serial_magnet'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install serial-magnet
